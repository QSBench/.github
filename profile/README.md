# QSBench

**Open-source generator of synthetic quantum datasets** for Quantum Machine Learning, noise robustness, error mitigation, and sim-to-real research.

We create clean, reproducible, richly annotated quantum circuit datasets with:
- Ideal and noisy expectation values
- Advanced realistic noise models (thermal relaxation, readout, device-like, etc.)
- Rich circuit metrics and metadata
- Deterministic train/val/test splits

### What we provide

- **Generator** — [`QSBench-Generator`](https://github.com/QSBench/QSBench-Generator) — tool to create your own datasets
- **Public Datasets** — ready-to-use high-quality quantum benchmarks on [Hugging Face](https://huggingface.co/QSBench)
- **Demos** — Thermal, Device, Readout, Amplitude, Depolarizing and more
